## Aplicação

Link para acesso em: [https://teste-tecnico-nology-sxtn.onrender.com](https://teste-tecnico-nology-sxtn.onrender.com)

Funcionalidades:

- Cálculo de cashback por tipo de cliente (comum ou VIP) e valor da compra
- Histórico de consultas registrado por IP do usuário
- Backend em Python (API de cálculo) + Frontend com Tailwind CSS

## 🛠️ Tecnologias Utilizadas

- Python
- Django
- PostgreSQL
- JavaScript
- HTML & CSS
- Render (deploy)
